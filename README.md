# Online Event Management System

## Project Overview
A web-based platform designed to simplify the process of organizing, managing, and attending events.

## Tech Stack
- **Frontend:** React.js, Vite, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (with Mongoose)

## Project Status
- **Phase 1 (Frontend):** Completed (UI components, Navbar, Event cards, and Create Event form).
- **Phase 2 (Backend):** Planned (API setup, Authentication, and Server logic).
- **Phase 3 (Database):** Planned (Data modeling and Integration).

## Features
- User Authentication
- Event Creation
- Event Browsing
